HTML & CSS

It is the standard language used to create the structure of web pages. It acts like the skeleton of a website, defining elements such as headings, paragraphs, and links.

JavaScript (JS)

It is a scripting language used to add interactivity to web pages. It allows you to create dynamic content like image sliders, pop-up windows, and real-time updates without refreshing the page.

Python

It is a high-level, easy-to-read programming language known for its simplicity. It is widely used in web development (backend), data science, artificial intelligence, and automation.

C++

It is a powerful, high-performance language that provides great control over system resources. It is commonly used for developing operating systems, game engines, and high-performance applications.
